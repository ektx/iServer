## iServer

iojs server  
- 是什么?  
本地前端开发迷你服务器,主要用于可以用模版的方式来开发你的静态页面  

- 功能  
用于开发预览网页,支持同网络下,多终端上页面查看  
用于生成模板合成的静态页面  
迷你文件服务器

## 使用方式:  
首先您要保证你的电脑上安装了 [IOJS](https://iojs.org/en/index.html) 或 [node.js](https://nodejs.org/)  

- 启动方式:
```sh
iojs epp.js
```
