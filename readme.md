## 功能说明:    
* Nodejs 文件服务器  
* 让开发者使用`ejs`和`jade`来组件化开发自己的静态页面     
* 支持样式的合并与压缩功能
* 生成文件统计与输出
* 覆盖生成功能    

## 更新日志
