## iServer

iojs server  
- 是什么?  
1.Nodejs 文件服务器  
2.让开发者使用`ejs`和`jade`来组件化开发自己的静态页面

## 使用方式:  
首先您要保证你的电脑上安装了 [IOJS](https://iojs.org/en/index.html) 或 [node.js](https://nodejs.org/)  

- 启动方式:
```sh
iojs epp.js
```
