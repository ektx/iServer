## iServer 2
![iServer 2](favicon.png 'Server 2.0')  

- 启动方式
```sh
node epp.js
```
 
### 功能说明:    
1. Nodejs 文件服务器  
2. 让开发者使用`ejs`和`jade`来组件化开发自己的静态页面     
3. 支持样式的合并与压缩功能
